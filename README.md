# The Total Forge

Production-ready React portfolio website for The Total Forge, built with Vite, Tailwind CSS, Framer Motion, React Router, and React Helmet Async.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Router
- React Helmet Async
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Project Structure

```text
src/
  components/
  data/
  pages/
```

## Notes

- Uses `@/` as an alias for `src/`
- Persists dark/light mode in `localStorage`
- Lazy-loads routes and portfolio imagery
